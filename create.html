Introduction

This game is a simple state machine and was built with vanilla HTML and Javascript. Javascript objects, loops, functions, and event handlers were used to dynamically generate the HTML for the different choices one might make.

Objects

In JavaScript, an object is a data structure that allows you to store and organize related data and functions. An object is a collection of properties, where each property consists of a key-value pair (a constant that defines the dataset, and its corresponding value). The key is always a string, but the value can be any JavaScript data type. An acceptable key value can stored as a:
  • String
  • Number
  • Array
  • Boolean
  • Null
  • Or even another object!
